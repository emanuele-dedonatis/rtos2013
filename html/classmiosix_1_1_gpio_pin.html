Detailed Description

This class allows to easiliy pass a Gpio as a parameter to a function. Accessing a GPIO through this class is slower than with just the Gpio, but is a convenient alternative in some cases. Also, an instance of this class occupies a few bytes of memory, unlike the Gpio class.
